# Vue 3 + TypeScript 学习路线

适合对象：没有前端基础、但有 Python 后端经验的开发者。

总目标：12 周左右建立 Vue 3 + TypeScript 的工程认知，并独立做出一个可演示、可复盘的小型前端项目。

建议投入：工作日每天 1.5 到 2 小时，周末 3 小时左右。

## 学习原则

1. 先理解浏览器、JavaScript 和 CSS，再学习 Vue。
2. 先学够用的 TypeScript，再进入组件化和工程化。
3. 每个阶段都要有可运行产物，不要只停留在看文档。
4. 练习优先选择你熟悉的后端业务场景，比如用户管理、任务列表、订单查询。
5. 学前端时把重点放在“状态如何变化、页面如何响应”，而不是死记 API。

## 推荐练习顺序

1. 理解当前项目入口与页面结构
2. 完成搜索、过滤、进度统计等小交互
3. 完成一个 Vue 版本的表单模块
4. 接入 Router、请求和 Pinia
5. 做一个用户管理或任务管理 mini admin

## 针对 Python 后端开发者的提醒

- 不要把重点放在“语法记忆”，而要放在“页面状态如何变化”。
- 不要一开始就追求 UI 库；先用原生 Vue 组件把数据流走通。
- 你的优势是数据结构、接口契约和错误处理，要把这个优势迁移到前端。
- 前端最容易卡住的点通常不是业务，而是组件边界、异步状态和调试方式。
