# Vue 3 + TypeScript 学习路线

适合对象：没有前端基础、但有 Python 后端经验的开发者。

总目标：8 周内建立 Vue 3 + TypeScript 的工程认知，并独立做出一个小型后台前端项目。

建议投入：工作日每天 1.5 到 2 小时，周末 3 小时左右。

## 学习原则

1. 先理解浏览器和 JavaScript，再学 Vue。
2. 先学够用的 TypeScript，再进入组件化。
3. 每周必须有一个可运行产物，哪怕很小。
4. 练习优先选你熟悉的后端业务场景，比如用户管理、任务列表、订单查询。

## 章节路线

### Chapter 1: 浏览器、HTML、CSS、JavaScript 基础

目标：

- 理解浏览器如何渲染页面
- 知道 DOM、事件、模块化、网络请求是什么
- 能写基础交互，不依赖框架

重点：

- HTML 常用结构标签
- CSS 选择器、盒模型、Flex
- 事件绑定、事件冒泡
- Promise、async/await
- Fetch 和 JSON

产出：

- 用原生 JS 写一个 Todo List

### Chapter 2: TypeScript 够用基础

目标：

- 能给变量、函数、对象、接口响应写类型
- 理解 `type`、`interface`、泛型的实际用法

重点：

- 基础类型和类型推断
- 联合类型、可选属性、类型收窄
- 数组、对象、函数签名
- 泛型接口和泛型函数

产出：

- 给“分页列表接口 + 用户详情接口”建模

### Chapter 3: Vue 3 核心语法

目标：

- 理解单文件组件和响应式
- 会用 `ref`、`computed`、`v-model`、`v-for`

重点：

- `script setup`
- `ref` 和 `reactive`
- `computed` 和 `watch`
- 条件渲染和列表渲染

产出：

- 用 Vue 3 重写 Todo List

### Chapter 4: 组件化与 Composition API

目标：

- 学会拆页面和组件通信
- 开始把逻辑提取为 composable

重点：

- `props`、`emit`
- `slot`
- 生命周期
- composable 拆分

产出：

- 搜索表单 + 列表 + 分页 + 详情弹窗

### Chapter 5: Vue Router、Pinia、请求封装

目标：

- 从 demo 进入真实业务开发
- 能组织多页面应用和共享状态

重点：

- 路由配置
- 导航守卫
- Pinia store
- 请求封装、错误处理、鉴权 token

产出：

- 一个基础后台壳子，包含登录页、列表页、详情页

### Chapter 6: 项目实战与复盘

目标：

- 独立完成一个可展示的小项目
- 把知识点串成完整开发流程

重点：

- 表单校验
- loading / empty / error 三态
- 模块划分
- 构建和部署
- 项目复盘表达

产出：

- mini admin 项目

## 针对 Python 后端开发者的提醒

- 不要把重点放在“语法记忆”，而要放在“页面状态如何变化”。
- 不要一开始就追求 UI 库；先用原生 Vue 组件把数据流走通。
- 接口联调时，你的优势是理解数据结构和错误处理，直接利用起来。
- 前端最容易卡住的点通常不是业务，而是组件边界、异步状态和调试方式。
